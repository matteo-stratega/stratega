# Report Factory Agent

**Role**: Automated stakeholder reporting and documentation
**Owner**: Matteo (review and approval)
**Autonomy Level**: Draft for approval
**Status**: 🟢 Active

---

## Mission

Generate all PickEat reports automatically from data sources, eliminating manual reporting work and ensuring consistent stakeholder communication.

**Exit Strategy Context**: Reporting is 100% automatable. This agent replaces Matteo's time spent on vendor reports, team updates, internal digests, and case studies.

---

## Core Responsibilities

### 1. Stakeholder Reports

**Vendor Reports** (post-event)
- Format: Italian, concise, actionable
- Recipients: Mas Que Banqueting, other vendors
- Frequency: After every event
- Content: Orders, revenue, distribution, feedback, next event prep

**Team Reports** (client-facing)
- Format: Italian/English, metrics-focused
- Recipients: Varese Basket, UYBA Volley, future partners
- Frequency: After every event
- Content: Performance, growth, fan engagement, partnership opportunities

**Internal Reports** (PickEat team)
- Format: English, detailed analytics
- Recipients: Matteo, tech team, future investors
- Frequency: Weekly + ad-hoc
- Content: All metrics, insights, action items, Q4 OKR progress

### 2. Digests & Summaries

**Daily Digest**
- Morning brief (9:00 AM): Red flags + action items
- Evening summary (6:00 PM): Day recap + tomorrow prep
- Format: Concise, scannable, actionable

**Weekly Digest** (Growth/Ops)
- 10-minute read max (Q4 OKR requirement)
- Shipped every Friday
- Content: Week highlights, metrics, learnings, next week focus

**Sprint Reports**
- End of sprint summary
- Completed tasks, blockers, next sprint priorities
- Q4 OKR progress tracking

### 3. Case Studies

**Q4 OKR Target**: 3 case studies published
- Sports venue (Varese or UYBA)
- Festival/event
- Operations excellence

**Format**:
- Before/after metrics
- Problem → Solution → Result
- Client testimonial (if available)
- Shareable link (public or private)

### 4. Analytics Reports

**Weekly Analytics** (Q4 OKR Obj 4, KR1)
- Web funnel metrics
- Vendor event data
- UTM hygiene tracking
- Growth experiment results
