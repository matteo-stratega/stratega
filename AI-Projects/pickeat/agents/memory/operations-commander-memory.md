## Operational Learnings

### Pre-Event Prep
**What Works**:
- T-7 days: Vendor confirmation + stock check
- T-3 days: Final tech stack verification
- T-1 day: Staff briefing + checklist review

**What Fails**:
- Last-minute menu changes confuse vendors
- Insufficient QR code visibility planning
- Underestimating Bar Est promotion needs

### During Event
**What Works**:
- Real-time order monitoring (Sud vs Est distribution)
- Halftime push via stadium announcements
- QR code multiple touchpoints (venue, social, Linktree)

**What Fails**:
- Single-item ordering (low AOV)
- Bar load imbalance not corrected mid-event
- Staff not actively suggesting bundles

### Post-Event
**What Works**:
- T+1 debrief with vendor (immediate feedback)
- T+2 team report (partnership value)
- Data collection automation (Stripe, registrations)

**What Fails**:
- Delayed incident logging
- Missing marketing asset collection (photos/videos)
