# Session Start

Quando l'utente esegue /start, fai questo:

## Step 1: Leggi il contesto
1. Leggi il file `brain/context.md`
2. Leggi l'ultimo file in `notes/daily-summaries/` (se esiste)

## Step 2: Riassumi
Fai un riassunto breve (max 5 punti):
- Cosa e' stato fatto l'ultima volta
- Cosa e' ancora da fare
- Su cosa ci stiamo concentrando

## Step 3: Chiedi conferma
Chiedi: **"Confermi queste priorita o vuoi cambiare qualcosa?"**

## Step 4: Aspetta
**NON fare altro. Aspetta che l'utente risponda.**
