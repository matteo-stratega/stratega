# Session Close

Quando l'utente esegue /close, fai questo:

## Step 1: Scrivi il report
Crea un file in `notes/daily-summaries/` chiamato `closing-[DATA].md` con:

```
# Closing [DATA]

## Riassunto
- **Fatto:** [cosa abbiamo completato]
- **Da fare:** [cosa resta]
- **Prossimo passo:** [la prima cosa da fare domani]

## File modificati
- [lista dei file creati o modificati]

## Note
[altre cose importanti da ricordare]
```

## Step 2: Aggiorna il contesto
Se ci sono novita importanti, aggiorna `brain/context.md`:
- Nuove decisioni
- Cambi di priorita
- Task completati

## Step 3: Conferma
Di all'utente: "Sessione chiusa! Report salvato in [nome file]"

**Fai tutto senza chiedere conferma.**
