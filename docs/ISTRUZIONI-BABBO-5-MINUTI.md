# Setup Claude - 5 Minuti

## 1. Installa Node.js
- Vai su **nodejs.org**
- Clicca il bottone verde "LTS"
- Installa (Next, Next, Next...)

## 2. Installa Claude Code
- Apri il terminale:
  - Windows: `Win+R` → scrivi `cmd` → Invio
  - Mac: cerca "Terminal"
- Copia e incolla:
```
npm install -g @anthropic-ai/claude-code
```

## 3. Crea cartella e metti il file
- Crea una cartella `workspace` in Documenti
- Metti dentro il file **SETUP-AUTOMATICO.md** (te lo mando)

## 4. Avvia Claude
Nel terminale:
```
cd Documents/workspace
claude
```
(Windows: `cd Documents\workspace`)

Al primo avvio fai login con account Anthropic.

## 5. Fai fare tutto a Claude
Scrivi:
```
Leggi SETUP-AUTOMATICO.md e fai tutto tu
```

**Fatto!** Claude configura tutto.

---

## Uso quotidiano
- `/start` = inizia lavoro
- `/close` = finisci e salva

## Obsidian (opzionale)
Installa da obsidian.md e apri la cartella workspace come "vault".
Vedi tutti i file in modo carino.
