# Pipeline Ideas

## Content Distribution System
**Added:** 12 Dec 2025
**Source:** Tool Friday Obsidian session

**Problem:**
Pubblicare contenuti su più piattaforme richiede adattamento manuale per ogni canale (LinkedIn, YouTube, Twitter, Newsletter, etc.)

**Solution idea:**
Sistema che prende un contenuto master e genera versioni ottimizzate per ogni piattaforma:
- LinkedIn: formato post + carosello
- YouTube: description + tags
- Twitter/X: thread + quote tweets
- Newsletter: embedded + CTA
- Instagram: caption + hashtags

**Trigger:** Video pubblicato su LinkedIn, serve repurposing automatico

**Priority:** Medium
**Effort:** Medium (workflow n8n + template library)
